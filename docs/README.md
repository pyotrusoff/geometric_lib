# Functions
## Circle
- Returns area and perimeter of given circle

## Square
- Returns area and perimeter of given square

## Rectangle
- Returns area and perimeter of given rectangle

# Triangle
- Returns area and perimeter of given triangle


# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a
